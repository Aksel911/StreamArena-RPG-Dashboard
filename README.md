# StreamArena RPG Dashboard

A feature-rich web dashboard for [StreamArena RPG](https://streamarenarpg.com) that lets you manage your inventory, equipment, guild, market listings, and more — all through a clean, dark-themed interface.

The dashboard communicates with the game's official API and uses a **disk‑based cache** to store large user data (udata) – bypassing Flask’s 4 KB cookie limit.

> **Note:** This project was generated almost entirely by AI (Claude Sonnet) as a demonstration of modern Flask + vanilla JS development.

---

## ✨ Features

- **Login** – Enter your game token once; session is kept on the server.
- **Dashboard** – Overview of character stats, gold, inventory slots, mail preview, daily spin timer, chests, and a voucher redeemer.
- **Inventory** – View, lock/unlock, destroy, or list items on the market. Mass‑delete unlocked items. Sort/filter by slot, power, name, or class.
- **Equipment** – See equipped items with power bars, unequip, or equip items directly from your inventory. Total power calculation.
- **Marketplace** – Browse all listings with live sorting by power, price, or value. Filter by slot/class. View your own listings and game chests. Buy items instantly.
- **Guild** – View guild info, roster (sorted by level), and guild chat. Send messages. Manual guild ID override if not auto‑detected.
- **Cosmetics** – See owned backpacks and browse the skin/backpack shops.
- **Bosses & Dungeons** – Live view of active world bosses (with HP bars), dungeons, and world battles.
- **Leaderboard** – Top 25 players (by class) and extended rankings.
- **Global Chat** – Read and send messages.
- **Mail** – Read your in‑game mail.
- **Disk Cache** – `udata` (~14 KB) is stored in `/tmp/streamarena_cache` for 5 minutes – no more cookie overflow.
- **Responsive** – Works on desktop and mobile (sidebar collapses).

## 🛠️ Technologies

- **Backend:** Python 3, Flask
- **Frontend:** Vanilla JavaScript, HTML5, CSS3 (no frameworks)
- **API:** REST calls to `streamarenarpg.com`
- **Caching:** Local JSON files with SHA‑256 hashed filenames
- **Fonts:** Google Fonts (Cinzel, Crimson Text, JetBrains Mono)

## 🚀 Installation & Setup

### 1. Clone the repository
```bash
git clone https://github.com/yourusername/streamarena-dashboard.git
cd streamarena-dashboard
```

### 2. Create a virtual environment (optional but recommended)
```bash
python -m venv venv
source venv/bin/activate   # Linux/Mac
venv\Scripts\activate      # Windows
```

### 3. Install dependencies
```bash
pip install flask requests
```

### 4. Run the application
```bash
python app.py
```

The server will start at `http://127.0.0.1:5000`.

## 🔑 Obtaining Your Game Token

1. Open the [StreamArena Portal](https://streamarenarpg.com/portal/app/portal.html) in your browser.
2. Open Developer Tools (F12) → Network tab.
3. Perform any action that calls the API (e.g., clicking on your character).
4. Look for a request to `portal_api.php` and copy the `token` parameter from the request payload.
5. Paste the token on the login page of the dashboard.

## ⚙️ Configuration

- **Cache directory:** `/tmp/streamarena_cache` (Linux/macOS). On Windows it will be created in the current drive's temp folder.
- **Cache TTL:** 300 seconds (5 minutes) – can be changed in `app.py` (`CACHE_TTL`).
- **Secret key:** Hardcoded in `app.py` (`streamarena_secret_2026`) – change it for production.
